Trailcam dashboard 
==================

The front-end Angular 5 application for NextGIS Web extension for trail cameras

## How to run

1. Install all dependencies from `trailcam-dashboard` directory:  
`npm install`

2. Running dev server:  
`npm start`  
or building production version:  
`npm build`
